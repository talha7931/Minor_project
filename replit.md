# Minor Project

## Overview
A minimal project imported from GitHub. Currently contains a simple static HTML page served via Python's built-in HTTP server.

## Setup
- **Frontend:** Static HTML (`index.html`) served on port 5000
- **Server:** Python `http.server` module (no external dependencies)
- **Workflow:** "Start application" — runs `python3 -m http.server 5000 --bind 0.0.0.0`

## Project Structure
- `index.html` — Main page
- `README.md` — Project title
- `.replit` — Replit configuration (Nix channel: stable-25_05)
