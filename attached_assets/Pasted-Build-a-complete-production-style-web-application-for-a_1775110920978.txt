Build a complete production-style web application for a residential society gate entry system using ANPR (Automatic Number Plate Recognition).

Project context:
- This project is for a society/colony gate entrance system using ANPR technology.
- The app must be compatible with Raspberry Pi and Raspberry Pi Camera modules.
- Use Raspberry Pi camera support through Picamera2/libcamera, not old legacy camera code.
- The app should also run in development on a normal computer with a webcam or uploaded test images.
- I am using Replit, so make the app easy to run there first, then portable to Raspberry Pi.
- Keep the system practical for a college project, but make the architecture realistic and complete.

Core goal:
Create a full-stack ANPR gate management app that:
1. Captures live frames from Raspberry Pi Camera or webcam.
2. Detects the vehicle number plate.
3. Runs OCR on the plate.
4. Validates and formats the plate text.
5. Checks the plate against an authorized residents/vehicles database.
6. Logs every entry/exit event.
7. Shows role-based dashboards.
8. Simulates or controls gate open/close actions.
9. Supports alerts for unauthorized vehicles.
10. Works efficiently on Raspberry Pi, so keep the design lightweight and optimized.

Important dashboard limit:
- Do NOT create too many dashboards.
- Maximum 3 dashboards only.
- Use exactly these 3 dashboards:
  1. Admin dashboard
  2. Security dashboard
  3. Resident dashboard

Users and dashboard responsibilities:

1) Admin dashboard:
- For society committee/admin/management
- Manage residents
- Manage authorized vehicles
- View all logs
- Configure ANPR settings
- Configure gate rules
- Manage users and roles
- View analytics, daily entries, denied attempts, camera health, and system status
- Export logs as CSV
- Review flagged OCR mismatches or low-confidence reads

2) Security dashboard:
- For gate guards/security staff

- Button to manually allow entry

- Show alerts for blacklisted or unknown vehicles
- Designed for fast use with minimal clicks

3) Resident dashboard:
- For colony residents
- Login to view their registered vehicles
- Add vehicle request or update request
- See own entry/exit history
- Request temporary visitor pass
- View visitor approvals and expiry
- View alerts if their vehicle was denied or mismatched
- No heavy analytics here; keep it simple
